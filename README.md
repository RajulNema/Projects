📂 Projects Overview
Below is a summary of the projects included in this repository. Each project demonstrates different skills, including Exploratory Data Analysis (EDA), Hypothesis Testing, and SQL.

1. Yulu - Hypothesis Testing
  File: Yulu - Hypothesis Testing.ipynb_Rajul.pdf

  Description: A statistical analysis project focusing on identifying the significant variables that affect the      demand for Yulu's shared electric cycles.

  Key Skills: Hypothesis Testing (T-test, ANOVA, Chi-square), Data Visualization, Statistical Inference.

2. Target - SQL Business Case
  File: Target_Businesscase_Rajul_SQLProject.pdf

  Description: An extensive SQL-based project analyzing customer purchasing behavior, order details, and regional   performance for Target to drive business strategies.

Key Skills: SQL (Joins, Subqueries, Window Functions, Aggregations), Data Extraction, Relational Databases.

3. Netflix - Business Case
  File: Business Case_ Netflix - Rajul Nema.ipynb.pdf

  Description: An exploratory data analysis of Netflix's content catalog. This project uncovers trends in content    production, geographical reach, and genre popularity to inform content strategy.

  Key Skills: Python (Pandas, Matplotlib, Seaborn), EDA, Data Cleaning, Trend Analysis.

4. Walmart - Business Case
  File: Business Case_ Walmart.ipynb - Rajul.pdf

  Description: A detailed business case study analyzing Walmart's sales data to identify top-performing branches,    customer behavior patterns, and factors influencing revenue.

  Key Skills: Python, EDA, Business Strategy Formulation, Consumer Behavior Analysis.

5. Aerofit - Data Analysis Project
  File: Aerofit_Rajul_Project.pdf

  Description: A customer profiling project for Aerofit treadmills. This analysis uses descriptive statistics and    probability to create customer profiles for different treadmill models and recommend data-driven marketing         strategies.

  Key Skills: Descriptive Statistics, Conditional Probability, Customer Profiling, Marketing Analytics.
